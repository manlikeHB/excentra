# Excentra — Frontend

The trading interface for Excentra Exchange. Built with Next.js, TypeScript, and Tailwind CSS.

For backend setup and full system architecture, see the [root README](../README.md).

---

## Prerequisites

- [Node.js](https://nodejs.org/) v20+
- [pnpm](https://pnpm.io/)

---

## Setup

**1. Install dependencies**

```bash
pnpm install
```

**2. Configure environment**

```bash
cp .env.example .env.local
```

`.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5098/api/v1
NEXT_PUBLIC_WS_URL=ws://localhost:5098/ws
```

**3. Start the dev server**

```bash
pnpm dev
```

Available at `http://localhost:3000`.

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start development server |
| `pnpm build` | Production build |
| `pnpm start` | Start production server |
| `pnpm lint` | Run ESLint |
| `pnpm type-check` | Run TypeScript compiler check |
| `pnpm gen:api` | Regenerate API types from OpenAPI spec |

---

## Regenerating API Types

API types are auto-generated from the backend's OpenAPI spec. After making backend changes, run:

```bash
# From the repo root — requires the backend to be running
pnpm --filter excentra-frontend gen:api
```

Or from the `frontend/` directory:

```bash
pnpm gen:api
```

This regenerates `src/lib/generated/api-types.ts`. Do not edit this file by hand.

## Project Structure

```
src/
├── app/                  # Next.js app router pages
│   ├── trade/            # Main trading interface
│   ├── dashboard/        # Portfolio, orders, settings
│   ├── admin/            # Admin panel
│   ├── login/
│   ├── register/
│   ├── forgot-password/
│   └── reset-password/
├── components/
│   ├── trading/          # OrderBook, OrderForm, PriceChart, etc.
│   ├── dashboard/        # Portfolio cards, asset management
│   ├── admin/            # UsersTable, PairsManager, MetricCard
│   ├── orders/           # OrdersTable, TradeHistoryTable
│   └── shared/           # Navbar, Skeleton, etc.
├── hooks/                # useOrderBook, useRecentTrades, usePairs, etc.
└── lib/
    ├── api.ts            # Typed API client
    ├── context.tsx       # Auth + WebSocket providers
    ├── types.ts          # Re-exports from generated types
    ├── symbols.ts        # Formatting helpers (formatPrice, formatDecimal)
    └── generated/
        └── api-types.ts  # Auto-generated — do not edit
```
